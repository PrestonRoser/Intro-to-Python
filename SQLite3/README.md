# DatabaseSerivce - Player Database CLI

This repository is a compilation of my work in databases, specifically a player database CLI written in Python and using SQLite3.

A simple Python backend practice project using SQLite.

The program allows users to:

- Insert player records
- Read players
- Search players by attributes

## Tech Stack

- Python3
- SQLite3
- CLI interface

## Run

Activate virtual environment:
`source .venv/bin/activate`

Run the program:
`python src/main.py`
